## Evolution summary

- Version 1.x focuses on program display and calorie estimation.
- Version 2.x introduces persistence, client progress tracking, and richer admin tooling.
- Version 3.x expands into role-based usage, reporting, workouts, and broader gym operations.
- The current repository adapts the concept into a Flask web application that is easier to containerize, test, and deploy through CI/CD tooling.
